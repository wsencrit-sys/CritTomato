# CritTomato
# 一个支持多时区选择和番茄钟功能的桌面时钟
# 前言：鄙人是重度AI使用者，桌面常年太平洋时区、TUN模式的科学上网，能找到市面上的桌面时钟无法自定义时区。为了模拟海外桌面环境的同时看本地时间方便，自己做了一款。
# 主要特点：
# 1.支持多时区切换
# 2.功能和视觉元素都极简风格
# 3.支持开机启动
# 4.可以进行一些简单的视觉设置
# 5.自定义番茄钟时长和提示语
# 6.专注和休息都有铃声
# 7.支持中英文
# 8.支持桌面置顶
# 总结：代码小白，自用小工具，可能不是太完善，后续会不定时更新，想用的小伙伴可以拿去玩，纯兴趣分享。
